# Kinesis Example

## Install Dependencies
```shell
> npm i
```

## Run Simple Producer
```shell
> node simple_producer.js
```
__Note__: Excepts a Kinesis Data Stream named `demo-test` with one shard.

## Run Simple Consumer
```shell
> node simple_consumer.js
```
__Note__: Excepts a Kinesis Data Stream named `demo-test` with one shard.
